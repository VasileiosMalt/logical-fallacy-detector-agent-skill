# Argument from fallacy

> **Category:** Formal Fallacy
> **Subcategory:** Formal

## Definition

The assumption that, if a particular argument for a conclusion is fallacious, then the conclusion by itself is false.

## Also Known As

- Fallacy fallacy
- Argumentum ad logicam
- Bad Reasons Fallacy
