# Loaded question

> **Category:** Informal Fallacy
> **Subcategory:** Improper Premise

## Definition

Asking a question that has an assumption built into it so that it can't be answered without appearing guilty.

## Also Known As

- Complex question
- Plurium interrogationum
- Trick question
