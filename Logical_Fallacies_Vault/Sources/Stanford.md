# Stanford University - Logical Fallacies

A Stanford-hosted reference drawing from Stephen Downes' well-regarded guide to logical fallacies.

**URL:** https://web.stanford.edu/~jonahw/PWR1/LogicalFallacies.htm

**Coverage:**
- Fallacies of distraction
- Appeals to motives
- Changing the subject
- Inductive fallacies
- Fallacies involving statistical syllogisms
- Causal fallacies
- Missing the point
- Fallacies of ambiguity
- Category errors
- Non sequitur
- Syllogistic errors
