# Wikipedia - List of Fallacies

An extensive structured list covering formal fallacies, informal fallacies, propositional fallacies, quantification fallacies, syllogistic fallacies, and red herring fallacies.

**URL:** https://en.wikipedia.org/wiki/List_of_fallacies

**Coverage:**
- Formal fallacies
- Propositional fallacies
- Quantification fallacies
- Formal syllogistic fallacies
- Informal fallacies (improper premise, faulty generalizations, questionable cause, relevance fallacies, red herring fallacies)
